# FindRESTUrl

same tool of https://github.com/mrmanzhaow/RestfulToolkit
功能跟RestfulToolkit类似 第一次搜索可能比较慢
代码是开源的

搜索java代码里面的REST URL

1.使用 `ctrl+\` 快捷键弹出搜索框 剪贴板有值可以直接进入搜索框
2.输入要搜索的URL片段
3.点击搜索结果跳转到对应的代码

使用场景: 我知道某个URL被调用了 但是不知道具体实现代码在哪里 因为URL可能是controller和method里面的mapping拼接的 找起来很麻烦
这时候就可以用这个插件进行快速搜索
